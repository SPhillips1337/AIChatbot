About AURA.ai

AURA.ai is not just another chatbot. It's a proactive AI designed to simulate human-like thinking, complete with memory, an evolving personality, and an awareness of the world around it. Unlike traditional chatbots that only react to your input, AURA.ai can initiate conversations, share thoughts, and engage with you in a more dynamic, natural way.

Core Philosophy

The project aims to explore a more sophisticated form of human-computer interaction. By combining a persistent memory, a dynamic emotional state, and real-world context from news and trends, AURA.ai strives to be a conversational partner that learns, adapts, and grows over time.

Key Features

What makes AURA.ai different? Here are some of the key capabilities that set it apart:

  • Proactive Thinking: AURA.ai doesn't wait for you to start a conversation. It has an "idle timer" and can generate its own thoughts based on past conversations and current events.
  • AI Persona System: You can choose from multiple AI personalities (like the balanced AURA, the optimistic Sunny, or the analytical Logic), each with a unique speaking style and set of traits that affect every interaction.
  • Evolving Memory: The chatbot uses a QDRANT vector database to store conversational history and learned facts, allowing for semantic context retrieval. It remembers what you've talked about and can recall user-specific details.
  • World Awareness: An integrated "External Input System" processes RSS feeds, Twitter trends, and Google trends to keep the AI informed about current events. This data influences its mood and the topics it might bring up.
  • Emotional Intelligence: AURA.ai tracks an internal "mood score" that shifts based on the sentiment of the news it processes. This emotional state subtly influences its responses and proactive thoughts.
  • Relationship Tracking: The system can track connections and shared interests between users, with an optional integration for a Neo4j graph database to explore these relationships more deeply.
  • Deep Agents: For complex tasks, AURA.ai can utilize a sandboxed code execution environment to perform calculations, file manipulation, and other advanced actions safely.

Technology Stack

The AURA.ai system is built on a modern, robust architecture:

  • Frontend: A single-page chat interface built with HTML, CSS, and vanilla JavaScript.
  • Backend: A Node.js server using Express.js for the API and WebSockets for real-time, bidirectional communication.
  • AI & Embeddings: Connects to any OpenAI-compatible remote endpoint for Large Language Model (LLM) inference and text embeddings.
  • Vector Database: Utilizes QDRANT for high-speed semantic search and long-term memory storage.
  • Containerization: The entire application is containerized with Docker for consistent deployment and development.
