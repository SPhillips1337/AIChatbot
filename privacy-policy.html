Privacy Policy

Last updated: December 6, 2024

Important: AURA.ai is an experimental AI system that learns from conversations. By using this service, you consent to data collection and processing as described below.

1. Data We Collect

  • Conversation Data: All messages you send and receive
  • Personal Facts: Information you share (name, preferences, interests, etc.)
  • Behavioral Data: Conversation patterns, response times, interaction frequency
  • Technical Data: IP address, browser type, session data

2. How We Use Your Data

  • Provide personalized AI responses
  • Remember context across conversations
  • Improve AI personality and responses
  • Generate proactive thoughts and questions

3. Data Storage

  • Data is stored in vector databases and JSON files
  • Conversations are embedded and stored indefinitely
  • Personal facts are stored in user profiles
  • Optional Neo4j graph database for relationship mapping

4. Your Rights (GDPR)

  • Access: Request a copy of your data via the profile interface
  • Rectification: Correct inaccurate facts via the profile interface
  • Erasure: Delete specific facts or request full account deletion
  • Portability: Export your data in JSON format
  • Withdraw Consent: Stop using the service to halt further data collection

5. Data Sharing

We do not share your personal data with third parties. Data is processed locally on our servers.

6. Contact

For privacy concerns or data requests, contact the system administrator.
